# Autonomous Cloud Cost Intelligence & Self-Optimizing Infrastructure Platform

## Overview
This project is an AI-powered cloud optimization platform that monitors cloud resource usage, predicts future costs using Machine Learning, and provides optimization recommendations.

- Frontend dashboard development started

- Machine Learning: Python, scikit-learn
